# Bible Grammar Reports — Index

This folder contains all research reports generated by the Berean Bible Bots project. Reports analyze the grammar, vocabulary, and structure of the Hebrew Old Testament, Aramaic portions of the OT, Septuagint (LXX), and Greek New Testament.
