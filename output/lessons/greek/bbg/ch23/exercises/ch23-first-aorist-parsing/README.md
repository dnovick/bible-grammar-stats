# Exercise: First Aorist Parsing Drill

**Chapter:** BBG Ch23 — First Aorist Active and Middle Indicative
**Type:** Morphology parsing drill
**Items:** 20

## Description

Twenty first aorist forms from common GNT verbs. Students parse each form (tense, voice, mood, person, number), provide the lexical form, and translate. Items include regular first aorists, stop-consonant combinations, liquid aorists, and contract verb aorists. Two second aorist forms are included as distractors.

## Coverage

| Form Type | Items |
|---|---|
| First aorist active — regular | 7 |
| First aorist active — stop consonant (ψ/ξ/σ) | 3 |
| First aorist active — liquid aorist | 3 |
| First aorist active — contract verb | 3 |
| First aorist middle | 2 |
| Second aorist distractors | 2 |
