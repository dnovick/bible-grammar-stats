# Exercise: Perfect Indicative Parsing Drill

**Chapter:** BBG Ch25 — Perfect Indicative
**Type:** Morphology parsing drill
**Items:** 20

## Description

Twenty perfect indicative forms from GNT vocabulary. Students parse each form (tense, voice, mood, person, number), identify the reduplication type, provide the lexical form, and explain the combinative aspect in their translation. Selected items include second perfects (no κ), perfect middle/passive forms, and aorist distractors that look similar.

## Coverage

| Form Type | Items |
|---|---|
| Perfect active — with κ formant | 8 |
| Perfect active — second perfect (no κ) | 3 |
| Perfect middle/passive | 5 |
| Theologically significant forms (γέγραπται, τετέλεσται, etc.) | 2 |
| Aorist distractors | 2 |
