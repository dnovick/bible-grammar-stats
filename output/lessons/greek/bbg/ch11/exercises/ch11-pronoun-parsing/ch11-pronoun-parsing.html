BBG Chapter 11 · First and Second Person Personal Pronouns

Parse each pronoun: Person (1st/2nd), Case (Nom/Gen/Dat/Acc), Number (Sg/Pl), and Translation.

Reminder: 1st person singular: ἐγώ / ἐμοῦ (μου) / ἐμοί (μοι) / ἐμέ (με). Plural: ἡμεῖς / ἡμῶν / ἡμῖν / ἡμᾶς. 2nd person singular: σύ / σοῦ (σου) / σοί (σοι) / σέ (σε). Plural: ὑμεῖς / ὑμῶν / ὑμῖν / ὑμᾶς. Accented forms are emphatic; unaccented (enclitic) are unemphatic.
