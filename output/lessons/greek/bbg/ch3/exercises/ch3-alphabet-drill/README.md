# Ch3 Alphabet Drill — Greek Letter Identification

## Description

A 24-item drill covering all letters of the Greek alphabet. Students identify the letter name and its pronunciation/sound from the written lowercase (and occasional uppercase) form. The drill reinforces both letter recognition and the phonetic value of each letter, including final sigma and the gamma nasal.

### Coverage table

| Item | Form | Answer |
|---|---|---|
| 1–24 | Lowercase Greek letter | Letter name + sound description |
