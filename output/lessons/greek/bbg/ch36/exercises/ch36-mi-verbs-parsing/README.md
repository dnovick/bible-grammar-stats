# μι-Verbs Parsing — Exercise

*BBG Chapter 36 · ἵστημι, τίθημι, δείκνυμι, and Related μι-Verbs*

---

## Description

A 20-item parsing drill on ἵστημι, τίθημι, δείκνυμι, ἀφίημι, ἀπόλλυμι, and οἶδα. Students parse each form (tense / voice / person / number / mood / lexical form) and translate. Items include the transitive/intransitive ἵστημι distinction, perfect-with-present-meaning (ἕστηκα, οἶδα), and common GNT contexts. An answer key is embedded in all formats.
