# Exercise: Imperfect Parsing Drill

**Chapter:** BBG Ch21 — Imperfect Indicative
**Type:** Morphology parsing drill
**Items:** 20

## Description

Twenty imperfect indicative forms. Students parse each form (tense, voice, mood, person, number), identify the augment type (syllabic or temporal), provide the lexical form, and translate with appropriate aspect language ("was ___-ing" or "used to ___").

## Coverage

| Form Type | Items |
|---|---|
| Imperfect active — regular (syllabic augment) | 8 |
| Imperfect active — vowel-initial (temporal augment) | 4 |
| Imperfect middle/passive | 4 |
| Imperfect of εἰμί | 2 |
| Compound verb (augment after prefix) | 2 |
