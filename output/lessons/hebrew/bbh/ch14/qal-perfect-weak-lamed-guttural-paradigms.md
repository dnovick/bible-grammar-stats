# Qal Perfect — III-ח/ע (Lamed-Guttural) Paradigms

*Full inflection tables for roots with a guttural at R3 (ח, ע) in the Qal Perfect.*

---

## III-ח: שָׁלַח (send)

| Person | Form | Notes |
|---|---|---|
| 3ms | שָׁלַ֫חַ | Patach furtive below final ח; pronounced [sha-ˈla-ħaʔ] |
| 3fs | שָׁלְחָה | Shewa under R1; patach before ח + ָה- |
| 2ms | שָׁלַחְתָּ | Patach furtive drops before suffix; ת is the suffix |
| 2fs | שָׁלַחְתְּ | |
| 1cs | שָׁלַחְתִּי | |
| 3cp | שָׁלְחוּ | Shewa under R1; guttural closes syllable before -וּ |
| 2mp | שְׁלַחְתֶּם | Both R1 and R2 reduce |
| 2fp | שְׁלַחְתֶּן | |
| 1cp | שָׁלַחְנוּ | Patach furtive drops before נוּ |

> **Key:** The patach furtive (פַּתַח גְּנוּבָה) appears in the 3ms and 3fs where the
> guttural stands at the end of the word in a word-final open syllable. It is
> **pronounced before the guttural** despite being written beneath it. Before consonantal
> suffixes (2ms, 1cs, etc.) the furtive does not appear.

## III-ע: שָׁמַע (hear)

| Person | Form | Notes |
|---|---|---|
| 3ms | שָׁמַ֫עַ | Patach furtive before ע |
| 3fs | שָׁמְעָה | Shewa under R1; patach + ע + ָה- |
| 2ms | שָׁמַעְתָּ | No furtive; suffix closes the syllable |
| 2fs | שָׁמַעְתְּ | |
| 1cs | שָׁמַעְתִּי | |
| 3cp | שָׁמְעוּ | Shewa under R1 |
| 2mp | שְׁמַעְתֶּם | Both reduce |
| 2fp | שְׁמַעְתֶּן | |
| 1cp | שָׁמַעְנוּ | |

## III-ח/ע: Strong Paradigm Comparison

| Person | Strong (קָטַל) | III-ח (שָׁלַח) | III-ע (שָׁמַע) |
|---|---|---|---|
| 3ms | קָטַל | שָׁלַ֫חַ | שָׁמַ֫עַ |
| 3fs | קָטְלָה | שָׁלְחָה | שָׁמְעָה |
| 2ms | קָטַלְתָּ | שָׁלַחְתָּ | שָׁמַעְתָּ |
| 2fs | קָטַלְתְּ | שָׁלַחְתְּ | שָׁמַעְתְּ |
| 1cs | קָטַלְתִּי | שָׁלַחְתִּי | שָׁמַעְתִּי |
| 3cp | קָטְלוּ | שָׁלְחוּ | שָׁמְעוּ |
| 2mp | קְטַלְתֶּם | שְׁלַחְתֶּם | שְׁמַעְתֶּם |
| 1cp | קָטַלְנוּ | שָׁלַחְנוּ | שָׁמַעְנוּ |

> **Note:** The 3ms and 3fs with patach furtive are the only cells that clearly distinguish
> this class from the strong paradigm. All other forms look essentially like strong roots.
