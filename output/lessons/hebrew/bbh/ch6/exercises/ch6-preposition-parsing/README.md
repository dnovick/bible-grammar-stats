# Ch6 Preposition Parsing Drill

25 Hebrew prepositional phrases. For each phrase: identify the preposition as it appears, give the base/dictionary form, explain any vowel change, identify the object noun, and translate the phrase.

## Coverage

| Part | Description | Items |
|---|---|---|
| Part A | Inseparable prepositions (בְּ לְ כְּ) — default sheva, two-sheva rule, composite sheva matching | 8 |
| Part B | Inseparable prepositions fused with the definite article — normal consonants and gutturals | 8 |
| Part C | מִן — independent form, prefixed with assimilation, compensatory lengthening before gutturals | 5 |
| Part D | Independent prepositions (אֶל, עַל, עִם) and the direct object marker אֵת | 4 |
