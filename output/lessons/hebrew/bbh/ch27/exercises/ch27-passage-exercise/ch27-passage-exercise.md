# "Spot the Hiphil" — Passage Exercise

*BBH Chapter 27 · Hiphil Weak Verbs*

---

## Instructions

Each item below shows a pointed Hiphil verb highlighted in its passage context. For each:

1. **Conjugation** — Perfect / Imperfect / Wayyiqtol / Weqatal / Imperative / Inf. Construct / Inf. Absolute / Participle
2. **PGN** — Person-Gender-Number (e.g., 3ms, 2mp, 1cs)
3. **Root** — three-letter root (lexical form)
4. **Weak class** — I-guttural / III-ח/ע / III-א / III-ה / I-נ / I-י / Biconsonantal / Strong
5. **Causative gloss** — a brief English rendering of the Hiphil meaning in context
