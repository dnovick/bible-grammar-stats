# "Vowel Identification" — Classification Drill

## Description

25-item drill covering all five Hebrew vowel classes, both types of simple sheva (vocal and silent), all three hatef (composite) shevas, matres lectionis, and dagesh forte. Students identify the vowel name, class, quantity, and any special features for each form.

### Coverage

| Category | Items | Count |
|---|---|---|
| A-class vowels | 1–5 | 5 |
| E-class vowels | 6–10 | 5 |
| I-class vowels | 11–13 | 3 |
| O-class vowels | 14–17 | 4 |
| U-class vowels | 18–19 | 2 |
| Simple sheva (vocal and silent) | 20–21 | 2 |
| Hatef (composite) shevas | 22–24 | 3 |
| Dagesh forte | 25 | 1 |
| **Total** | | **25** |
