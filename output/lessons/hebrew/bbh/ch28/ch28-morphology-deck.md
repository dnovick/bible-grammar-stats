# BBH Chapter 28 — Hophal Strong Morphology Flashcards

24 cards covering 3 roots across all Hophal conjugations, emphasizing the Hiphil/Hophal relationship.

**Roots:**
- **מות** (Hiphil: to put to death → Hophal: to be put to death) — 68 Hophal occurrences
- **נגד** (Hiphil: to tell, declare → Hophal: to be told, reported) — 35 Hophal occurrences
- **בּוּא** (Hiphil: to bring → Hophal: to be brought) — 24 Hophal occurrences

## Key Diagnostic Summary

| Card Front | Expected Back |
|---|---|
| What is the ONE universal Hophal marker? | u-class vowel (Shureq וּ or Qibbuts ֻ) under the prefix consonant |
| Hophal Perfect prefix | **הוּ** (he + Shureq) — e.g., הוּמַת |
| Hophal Imperfect prefix | **יוּ/תּוּ** (Shureq) — e.g., יוּמַת |
| Hophal Wayyiqtol prefix | **וַיּוּ/וַתּוּ** — e.g., וַיּוּמַת |
| Hophal Participle prefix | **מוּ/מֻ** — e.g., מוּמָת |
| Does Hophal have Imperative? | No — passive cannot be commanded |
| Hophal is passive of which stem? | **Hiphil** (causative active) |
| Most common Hophal form (99×)? | יוּמַת — "shall be put to death" |

## Hophal — מות (to be put to death)

| Front | Back |
|---|---|
| יוּמַת | מות Hophal+Imperfect+3ms — "shall be put to death" — most common Hophal form (99×); death penalty formula |
| יוּמְתוּ | מות Hophal+Imperfect+3mp — "they shall be put to death" |
| הוּמַת | מות Hophal+Perfect+3ms — "he has been put to death" (הוּ prefix = definitive Hophal Perfect marker) |
| וַתּוּמַת | מות Hophal+Wayyiqtol+3fs — "and she was put to death" (2Ki 11:16) |
| מוּמָת | מות Hophal+Participle+ms — "one who is to be put to death" — מוּ prefix + Qamets under R2 |
| הוּמֵת | מות Hophal+Inf.Absolute — Tsere under R2 (contrast Perfect Patach הוּמַת); rare |
| מוֹת יוּמַת | מות Inf.Abs.(Qal) + Hophal Imperfect — cognate absolute; "shall surely be put to death" |

## Hophal — נגד (to be told, reported)

| Front | Back |
|---|---|
| הֻגַּד | נגד Hophal+Perfect+3ms — "it was told" (הֻ = Qibbuts variant of Hophal prefix); Deu 17:4 |
| וַיֻּגַּד | נגד Hophal+Wayyiqtol+3ms — "and it was told" — standard narrative report formula (Gen 22:20; 27:42; etc.) |
| יֻגַּד | נגד Hophal+Imperfect+3ms — "it will be told" — Qibbuts (ֻ) = alternate Hophal prefix vowel |
| וְהֻגַּד | נגד Hophal+Weqatal+3ms — "and it will be told" — וְ + Hophal Perfect form; Jos 9:24 |

## Hiphil / Hophal Contrast Pairs

| Card Front | Expected Back |
|---|---|
| הֵמִית vs. יוּמַת | Hiphil הֵמִית = "he put to death" (active causative) / Hophal יוּמַת = "he was put to death" (passive causative) |
| הִגִּיד vs. הֻגַּד | Hiphil הִגִּיד = "he told/declared" / Hophal הֻגַּד = "it was told" |
| הֵבִיא vs. הוּבָא | Hiphil הֵבִיא = "he brought" / Hophal הוּבָא = "it was brought" |
| הִכָּה vs. מֻכִּים | Hiphil הִכָּה = "he struck/killed" / Hophal מֻכִּים = "those being struck" |
