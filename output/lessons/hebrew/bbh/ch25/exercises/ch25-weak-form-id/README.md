# "Weak-Form Identification Drill" — Exercise

*BBH Chapter 25 · Niphal Weak Verbs*

---

## Description

A 35-item identification and parsing exercise covering all five Niphal weak classes (III-א, III-ה, I-guttural, I-נ, I-י). Part A (items 1–25) groups forms by class — five items per class — so students can build class-by-class recognition before mixing. Part B (items 26–35) presents forms from all five classes in random order; students must identify the weak class first, then parse. Drawn primarily from Genesis and Exodus, with additional examples from prophetic texts. Answer keys for both parts are included, with brief explanatory notes in Part B. Four discussion questions focus on pairs of forms that could be confused.

*(Item 31 in Part B counts two words as one exercise item.)*
