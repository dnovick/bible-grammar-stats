# ch32-qal-piel-pual-contrast

**Chapter 32 — Qal–Piel–Pual Contrast Drill (Strong Roots)**

*BBH Chapter 32 · Pual Strong Verbs*

## Description

A 15-item triplet drill showing the same root in Qal, Piel, and Pual side by side. Students identify the stem and translate each form. The side-by-side format highlights the systematic R1 vowel shift: variable (Qal) → hiriq/tsere (Piel) → qibbuts/holem (Pual), with dagesh forte in R2 for both Piel and Pual.

Roots covered: ברך, שׁלח, דבר, כבד, ספר.

## Coverage

| Triplet | Root | Gloss |
|---|---|---|
| 1 | ברך | kneel / bless / was blessed |
| 2 | שׁלח | sent / sent away / was sent away |
| 3 | דבר | spoke (rare) / spoke / was spoken |
| 4 | כבד | was heavy / honored / was honored |
| 5 | ספר | counted / recounted / was recounted |
