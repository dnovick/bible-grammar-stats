# ch32-stem-id-drill

**Chapter 32 — Stem-ID Drill (Strong Roots)**

*BBH Chapter 32 · Pual Strong Verbs*

## Description

A 24-item drill mixing Qal, Piel, and Pual of strong roots (8 forms each), interleaved across multiple conjugations. Students identify the stem, parse fully, and translate. The drill emphasizes distinguishing the three stems through R1 vowel identification and the presence/absence of dagesh forte in R2.

Primary root: דבר (items 1–3, 10–12, 16–23). Secondary roots: שׁלח (4–6, 13–15), כבד (7–9), ספר (24).
