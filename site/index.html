About This Project

Berean Bible Bots is an open-source project that generates and maintains statistical analyses and study materials for Old Testament Hebrew/Aramaic and New Testament Greek. All exercises are grounded in the MACULA Hebrew WLC and Greek datasets, ensuring every form, gloss, and reference is drawn from real scripture data.

Materials are freely available under the MIT license. Source code and data scripts are on GitHub.
