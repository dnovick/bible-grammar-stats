# δίδωμι Indicative Parsing — Exercise

*BBG Chapter 34 · The Indicative Forms of δίδωμι and Its Compounds*

---

## Description

A 20-item parsing drill on the indicative forms of δίδωμι and its compounds (especially παραδίδωμι). Students parse each form (tense / voice / person / number / mood / lexical form) and translate. Items cover all indicative tenses (present, imperfect, aorist, perfect passive). An answer key is embedded in all formats.

### Coverage

| Tense/Voice | Count | Items |
|---|---|---|
| Present active indicative | 5 | #1–#5 |
| Imperfect active indicative | 3 | #6–#8 |
| Aorist active indicative | 5 | #9–#13 |
| Aorist passive indicative | 4 | #14–#17 |
| Present/Imperfect middle/passive | 3 | #18–#20 |
