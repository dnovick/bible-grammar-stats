# Exercise: Ch16 Present Active Indicative Parsing

**Chapter:** BBG Ch16 — Present Active Indicative
**Type:** Morphological parsing drill with translation
**Items:** 20

## Description

Parse each present active indicative verb form (tense, voice, mood, person, number, lexical form) and provide a translation. Verbs are drawn from the GNT and from BBG vocabulary learned through Ch16.

## Coverage

| Feature | Forms drilled |
|---|---|
| Person | 1st, 2nd, 3rd |
| Number | Singular and plural |
| Verbs | Regular thematic verbs (no contract verbs — see Ch17) |
