# Exercise: Ch18 Present Middle/Passive Indicative Parsing

**Chapter:** BBG Ch18 — Present Middle/Passive Indicative
**Type:** Morphological parsing drill with voice identification
**Items:** 20

## Description

Parse each present middle/passive indicative verb form (person, number, lexical form) and identify the most likely voice (Middle or Passive) based on context clues provided. Deponent verbs are included and labeled.

## Coverage

| Feature | Forms drilled |
|---|---|
| Person | 1st, 2nd, 3rd |
| Number | Singular and plural |
| Voice | Middle, Passive, Deponent |
| Verbs | λύω (paradigm), common deponents (ἔρχομαι, γίνομαι, πορεύομαι, etc.) |
