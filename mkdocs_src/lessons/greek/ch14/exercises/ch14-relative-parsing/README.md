# Exercise: Ch14 Relative Pronoun Parsing

**Chapter:** BBG Ch14 — Relative Pronoun
**Type:** Morphological parsing drill with translation
**Items:** 20

## Description

For each item: (1) parse the relative pronoun (gender, case, number), (2) identify the antecedent, and (3) translate the relative clause. Forms are drawn from GNT passages and constructed examples covering all genders, cases, and numbers.

## Coverage

| Feature | Forms drilled |
|---|---|
| Gender | Masculine, feminine, neuter |
| Case | Nominative, genitive, dative, accusative |
| Number | Singular and plural |
| Function | Subject, direct object, indirect object, genitive modifier, object of preposition |
