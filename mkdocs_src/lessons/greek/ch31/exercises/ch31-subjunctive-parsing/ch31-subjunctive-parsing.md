## Instructions

For each item:
1. Parse the **underlined** subjunctive form: tense / voice / person / number / mood / lexical form.
2. Identify the **use**: purpose (ἵνα), 3rd class conditional (ἐάν), hortatory, prohibition, indefinite, or emphatic denial.
3. **Translate** the sentence.
