# Exercise: Stem Change Drill

**Chapter:** BBG Ch20 — Verbal Roots (Patterns 2–4)
**Type:** Root identification and pattern recognition drill
**Items:** 20

## Description

Twenty verbs drawn from the GNT. For each present form, students identify (1) the underlying verbal root, (2) the pattern type (2, 3, or 4), and (3) give the future active (or future deponent) form. Verbs are drawn from Pattern 2 (stop-consonant changes), Pattern 3 (ablaut/nasal infix), and Pattern 4 (suppletive stems).

## Coverage

| Pattern | Items |
|---|---|
| Pattern 2 — Labial stop (ψ) | 4 |
| Pattern 2 — Velar stop (ξ) | 4 |
| Pattern 2 — Dental stop | 3 |
| Pattern 3 — Ablaut/Nasal infix | 5 |
| Pattern 4 — Suppletive | 4 |
