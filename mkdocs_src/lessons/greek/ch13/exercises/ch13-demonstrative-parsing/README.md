# Exercise: Ch13 Demonstrative Parsing

**Chapter:** BBG Ch13 — Demonstrative Pronouns/Adjectives
**Type:** Morphological parsing drill
**Items:** 20

## Description

Parse each demonstrative form (giving gender, case, number, and the lexical form) and identify whether it functions as an **adjective** (modifying a noun in predicate position) or a **pronoun** (substantival/standalone). GNT examples are drawn from the Synoptic Gospels and John.

## Coverage

| Demonstrative | Forms drilled |
|---|---|
| οὗτος | Masculine, feminine, neuter; all cases; singular and plural |
| ἐκεῖνος | Masculine, feminine, neuter; all cases; singular and plural |
| Use | Adjective (predicate position) and pronoun (substantival) |
