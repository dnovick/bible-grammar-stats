# Ch27 — Niphal–Hiphil Contrast Drill

*BBH Chapters 25 & 27 · Niphal and Hiphil Weak Verbs*

---

## Description

A 20-item drill focused on distinguishing Niphal from Hiphil forms in weak roots. Each item asks students to identify the stem, parse the conjugation and PGN, give the root and weak class, and translate in context. Forms are drawn from attested OT passages only.

The exercise is organized in three parts:
- **Part A** (items 1–8): Root pairs with contrasting Niphal and Hiphil meanings (שָׁמַע, מָצָא, גָּלָה/עָלָה, נָגַשׁ)
- **Part B** (items 9–15): I-י and Biconsonantal roots where both stems use similar prefix patterns (יָלַד ×3, יָרַד, יָדַע, קוּם, כּוּן)
- **Part C** (items 16–20): Mixed review from any weak class
