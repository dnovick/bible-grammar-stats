# ch33-qal-piel-pual-contrast

**Chapter 33 — Qal–Piel–Pual Contrast Drill (Weak Roots)**

*BBH Chapter 33 · Pual Weak Verbs*

## Description

A 15-item triplet drill showing the same weak root in Qal, Piel, and Pual side by side. Identical in format to the Ch32 contrast drill, but all roots are weak. Students identify the stem and translate each form, attending carefully to how weak-root changes (III-ה endings, guttural compensatory lengthening) interact with the Pual stem markers.

## Roots Covered

| Triplet | Root | Weak Type | Gloss |
|---|---|---|---|
| 1 | גלה | III-ה | went into exile / revealed / was revealed |
| 2 | כלה | III-ה | finished / completed / was consumed |
| 3 | ברך | R2 = ר (rejects dagesh) | knelt / blessed / was blessed |
| 4 | ענה | III-ה, I-Guttural (ע) | answered / afflicted / was afflicted |
| 5 | חיה | III-ה | lived / kept alive / was kept alive |
