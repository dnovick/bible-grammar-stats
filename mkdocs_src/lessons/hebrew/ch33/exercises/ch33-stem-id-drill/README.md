# ch33-stem-id-drill

**Chapter 33 — Stem-ID Drill (Weak Roots)**

*BBH Chapter 33 · Pual Weak Verbs*

## Description

A 24-item drill mixing Qal, Piel, and Pual of weak roots (8 forms each), interleaved across multiple conjugations. Students identify the stem, parse fully, and translate. Mirrors the Ch32 Stem-ID Drill in structure but applies the same skills to III-ה and guttural roots, where weak-root changes interact with stem markers.

Primary root: גלה (items 1–3, 10–12, 16–17, 19–23). Secondary roots: ברך (4–6, 13–14), כלה (7–9, 15), צוה (18, 24).

## Key Contrasts Practiced

| Contrast | Forms |
|---|---|
| Piel vs Pual imperfect (III-ה) | יְגַלֶּה vs יְגֻלֶּה |

| Qal vs Piel wayyiqtol | וַיִּגֶּל vs וַיְגַל |
