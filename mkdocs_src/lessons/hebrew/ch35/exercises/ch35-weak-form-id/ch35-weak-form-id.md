# "Weak-Form Identification Drill" — Exercise

*BBH Chapter 35 · Hithpael Weak Verbs*

---

## Instructions

**Part A:** Forms are grouped by weak class (5 per class, 4 groups = 20 items total). For each, identify: conjugation, PGN, root.
**Part B:** Forms are mixed. Identify: weak class first, then conjugation, PGN, root.

The Hithpael is identified by its הִתְ prefix (or וַיִּתְ in wayyiqtol) plus Dagesh Forte in R2. Weak roots cause predictable modifications, and one phonological rule is unique to the Hithpael: **sibilant metathesis**.

**Sibilant Metathesis Rule:** When R1 is a sibilant (שׁ / שׂ / ס / צ), the ת of the Hithpael prefix swaps position with R1:
- Root שָׁמַר: expected *הִתְשָׁמֵּר → becomes הִשְׁתַּמֵּר (ת and שׁ swap)
- Root שָׁחָה: expected *הִתְשַׁחֲוָה → becomes הִשְׁתַּחֲוָה
- Root צָרַר: expected *הִתְצַדֵּק → becomes הִצְטַדֵּק (ת → ט and swaps with צ)

---

## Weak Class Reference

| Class | Key diagnostic signal |
|---|---|
| **III-he** | Qamets + ה (perfect 3ms); ה dropped in wayyiqtol/jussive (apocopation); seghol + ה (imperfect/participle) |
| **I-guttural** | Hateph vowel under R1 in perfect/participle; composite shewa replaces simple shewa throughout |
| **Sibilant Metathesis** | הִשְׁתַּ / וַיִּשְׁתַּ prefix instead of הִתְ / וַיִּתְ — the two consonants after the prefix are R1 and ת, in reversed order |
| **I-nun** | Nun is retained in Hithpael (unlike Hiphil where nun assimilates); prefix pattern הִתְנַ is regular |

## Discussion Questions

**1.** Items 11–14: in each case, which letter is R1 and which is the Hithpael ת? Write out the expected (pre-metathesis) form of item 14, then the attested (post-metathesis) form, and label which consonants swapped.

**2.** Items 6–10 (I-guttural): the hateph vowel under the guttural means these forms look similar to Piel I-guttural forms. Write a test you could apply to distinguish a Hithpael I-guttural from a Piel I-guttural.

**3.** Compare item 1 (וַיִּתְגַּל, III-he Hithpael wayyiqtol) and item 11 (וַיִּשְׁתַּחוּ, III-he + sibilant Hithpael wayyiqtol). Both are apocopated forms. What is left of the root in each case? Is the apocopation the same process?

**4.** Item 20 (מִתְנַדְּבִים): the root is נָדַב (I-nun). In the Hithpael, the nun is NOT assimilated (compare: it IS assimilated in the Hiphil הִדְּיב). Explain why retention of the nun makes sense given the Hithpael prefix pattern הִתְנַ.

**5.** Item 23 (הִתְרוֹמֵם) involves a hollow or geminate root. What does the Hithpael of a hollow/geminate root look like? How does the holem-vav arise, and why does the expected Dagesh Forte in R2 not appear as a visible dagesh?
