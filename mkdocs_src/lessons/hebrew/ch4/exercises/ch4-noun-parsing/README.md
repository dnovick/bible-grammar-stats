# Ch4 Noun Parsing Drill — Hebrew Nouns

Identify the gender, number, state, lexical form, and gloss for each of 25 Hebrew noun forms.

## Coverage

| Category | Description | Items |
|---|---|---|
| Segolate singulars and plurals | מֶלֶךְ, נֶפֶשׁ, אֶרֶץ, סֵפֶר | 1–3, 17–19, 22 |
| Feminine nouns (abs. and cstr.) | תּוֹרָה, מַלְכָּה, שָׁנָה | 4–5, 20, 24–25 |
| Masculine plurals and constructs | דָּבָר, בֵּן, יוֹם | 6–8, 16, 23 |
| Irregular plurals | אִישׁ/אֲנָשִׁים, אִשָּׁה/נָשִׁים, עִיר/עָרִים, בַּיִת/בָּתִּים, בַּת/בָּנוֹת | 9–11, 14–15 |
| Duals | יָד, עַיִן, שָׁנָה | 12–13, 21 |
