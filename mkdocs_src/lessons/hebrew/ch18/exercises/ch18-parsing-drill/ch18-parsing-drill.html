Part C — Disambiguation: Imperative, Imperfect, or Jussive?

Key rule: Positive commands use the Imperative (no prefix). Negative commands never use the Imperative — they use אַל + Jussive (immediate prohibition) or לֹא + Imperfect (permanent prohibition).
