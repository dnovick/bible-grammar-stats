StemKey Marker
QalNo prefix; basic vowel pattern; no dagesh in R2; III-ה ends in ָה (qamets-he, no dagesh)
Niphalנִ prefix (perfect) or assimilated dagesh in R1 (imperfect); III-ה ends in ָה with chirik–qamets
Hiphilהִ prefix (perfect); יַ/תַ prefix (imperfect); III-ה lacks chirik-yod but keeps patach prefix
PielDagesh in R2 (or virtual doubling for gutturals); shewa prefix in imperfect; מְ prefix in participle; III-ה ends in ָּה (qamets + dagesh-he)
I-Guttural tip: Gutturals (ב, ח, ע, etc.) reject dagesh. For Piel, look instead for compensatory lengthening — the vowel before R2 is longer than expected (e.g., qamets instead of chirik, tsere instead of segol).
III-ה tip: Compare the ending — Qal/Hiphil perfect 3ms = ָה; Piel/Niphal perfect 3ms = ָּה (with dagesh or marks of Piel); Piel imperative/participle shows dagesh in R2 before the ה ending.
