# Niphal (נִפְעַל) — Full Conjugation Paradigms

*Model root: קטל (Qal: קָטַל). All forms are strong-root paradigms.*
*Qal column is included for direct comparison at each conjugation.*
*Source: Basics of Biblical Hebrew, Pratico & Van Pelt, Ch. 24*

---

## Perfect (Qatal)

*BBH §24.4 — Niphal prefix: נִ; characteristic vowel: patach under first root letter*

| Person | Qal | **Niphal** |
|---|---|---|
| 3ms | קָטַל | **נִקְטַל** |
| 3fs | קָטְלָה | **נִקְטְלָה** |
| 3cp | קָטְלוּ | **נִקְטְלוּ** |
| 2ms | קָטַלְתָּ | **נִקְטַלְתָּ** |
| 2mp | קְטַלְתֶּם | **נִקְטַלְתֶּם** |
| 2fs | קָטַלְתְּ | **נִקְטַלְתְּ** |
| 2fp | קְטַלְתֶּן | **נִקְטַלְתֶּן** |
| 1cs | קָטַלְתִּי | **נִקְטַלְתִּי** |
| 1cp | קָטַלְנוּ | **נִקְטַלְנוּ** |

> **Key marker:** נִ prefix + *patach* under first root consonant. Distinguished from Hiphil perfect (הִקְטִיל) by נִ vs. הִ and the absence of the characteristic *chiriq* in the second syllable.

## Imperfect (Yiqtol)

*BBH §24.6 — Niphal: dagesh forte in first root letter (assimilated נ) + tsere in final syllable*

| Person | Qal | **Niphal** |
|---|---|---|
| 3ms | יִקְטֹל | **יִקָּטֵל** |
| 3fs | תִּקְטֹל | **תִּקָּטֵל** |
| 3mp | יִקְטְלוּ | **יִקָּטְלוּ** |
| 3fp | תִּקְטֹלְנָה | **תִּקָּטַלְנָה** |
| 2ms | תִּקְטֹל | **תִּקָּטֵל** |
| 2fs | תִּקְטְלִי | **תִּקָּטְלִי** |
| 2mp | תִּקְטְלוּ | **תִּקָּטְלוּ** |
| 2fp | תִּקְטֹלְנָה | **תִּקָּטַלְנָה** |
| 1cs | אֶקְטֹל | **אֶקָּטֵל** |
| 1cp | נִקְטֹל | **נִקָּטֵל** |

> **Key marker:** Doubled (dagesh forte) first root consonant + *tsere* in final syllable. The doubling reflects assimilation of the נ prefix. Qal has *hiriq* prefix + *holem* final (יִקְטֹל); Niphal has same prefix vowel but dagesh in R1 and *tsere* final.

## Wayyiqtol

*Sequential narrative form of the imperfect. Same dagesh + tsere pattern as the yiqtol.*

| Person | Qal | **Niphal** |
|---|---|---|
| 3ms | וַיִּקְטֹל | **וַיִּקָּטֵל** |
| 3fs | וַתִּקְטֹל | **וַתִּקָּטֵל** |
| 3mp | וַיִּקְטְלוּ | **וַיִּקָּטְלוּ** |
| 3fp | וַתִּקְטֹלְנָה | **וַתִּקָּטַלְנָה** |
| 2ms | וַתִּקְטֹל | **וַתִּקָּטֵל** |
| 2mp | וַתִּקְטְלוּ | **וַתִּקָּטְלוּ** |
| 1cs | וָאֶקְטֹל | **וָאֶקָּטֵל** |
| 1cp | וַנִּקְטֹל | **וַנִּקָּטֵל** |

> **Key marker:** Both Qal and Niphal use וַיִּ prefix — the distinguishing feature is the dagesh forte in R1 in the Niphal (וַיִּקָּטֵל) vs. its absence in the Qal (וַיִּקְטֹל).

## Weqatal

*Sequential future/consequential form of the perfect. וְ + perfect form.*

| Person | Qal | **Niphal** |
|---|---|---|
| 3ms | וְקָטַל | **וְנִקְטַל** |
| 3fs | וְקָטְלָה | **וְנִקְטְלָה** |
| 3cp | וְקָטְלוּ | **וְנִקְטְלוּ** |
| 2ms | וְקָטַלְתָּ | **וְנִקְטַלְתָּ** |
| 2mp | וּקְטַלְתֶּם | **וְנִקְטַלְתֶּם** |
| 1cs | וְקָטַלְתִּי | **וְנִקְטַלְתִּי** |
| 1cp | וְקָטַלְנוּ | **וְנִקְטַלְנוּ** |

> **Key marker:** וְ + identical to the perfect paradigm. Context distinguishes it from a simple perfect with vav-conjunction.

## Imperative

*BBH §24.8 — Niphal prefix: הִ; tsere in final syllable*

| Form | Qal | **Niphal** |
|---|---|---|
| 2ms | קְטֹל | **הִקָּטֵל** |
| 2fs | קִטְלִי | **הִקָּטְלִי** |
| 2mp | קִטְלוּ | **הִקָּטְלוּ** |
| 2fp | קְטֹלְנָה | **הִקָּטַלְנָה** |

> **Key marker:** Niphal imperative has הִ prefix (*hiriq*) + dagesh in R1 + *tsere*. Qal has no prefix. Distinguish from Hiphil imperative (הַקְטֵל — *patach* under הַ, no dagesh in R1).

## Infinitive Construct

*BBH §24.10 — Niphal prefix: הִ; tsere in final syllable*

| Form | Qal | **Niphal** |
|---|---|---|
| Base | קְטֹל | **הִקָּטֵל** |
| With לְ | לִקְטֹל | **לְהִקָּטֵל** |
| With בְּ | בִּקְטֹל | **בְּהִקָּטֵל** |
| With כְּ | כִּקְטֹל | **כְּהִקָּטֵל** |

> **Key marker:** הִ prefix + dagesh in R1 + *tsere*. Identical in form to the Niphal imperative 2ms; context and prepositions clarify function.

---

## Infinitive Absolute

*BBH §24.12 — Niphal prefix: נִ; holem in final syllable*

| Form | Qal | **Niphal** |
|---|---|---|
| Standard | קָטוֹל | **נִקְטֹל** |

> **Usage:** Appears before or after a finite Niphal verb for emphasis (הִמָּצֵא יִמָּצֵא "he will certainly be found," Exo 22:3). Qal inf. absolute uses *qamets-holem* (קָטוֹל); Niphal uses נִ prefix + *holem* (נִקְטֹל).

---

## Participle (Passive)

*BBH §24.13 — Niphal prefix: נִ; qamets under second root consonant*

| Form | Qal | **Niphal** |
|---|---|---|
| ms | קֹטֵל | **נִקְטָל** |
| fs | קֹטֶלֶת | **נִקְטֶלֶת** |
| mp | קֹטְלִים | **נִקְטָלִים** |
| fp | קֹטְלוֹת | **נִקְטָלוֹת** |

> **Key marker:** נִ prefix + *qamets* under R2 (ms). Distinguish from the perfect 3ms (נִקְטַל — *patach* under R2 vs. *qamets* in the participle).

## Summary: Qal vs. Niphal at a Glance

| Conjugation | Qal marker | **Niphal marker** |
|---|---|---|
| Perfect | קָ / *patach-patach* | **נִ + *patach*** |
| Imperfect | יִקְטֹל / *holem* final | **dagesh in R1 + *tsere*** |
| Wayyiqtol | וַיִּקְטֹל / no dagesh in R1 | **וַיִּקָּטֵל / dagesh in R1** |
| Weqatal | וְקָ | **וְנִ** |
| Imperative | no prefix | **הִ + dagesh in R1 + *tsere*** |
| Inf. Construct | קְטֹל | **הִקָּטֵל** |
| Inf. Absolute | קָטוֹל | **נִקְטֹל** |
| Participle ms | קֹטֵל / no prefix | **נִקְטָל / נִ prefix + *qamets*** |
