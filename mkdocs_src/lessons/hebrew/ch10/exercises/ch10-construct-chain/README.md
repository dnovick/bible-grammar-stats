# Ch10 Construct Chain Drill

For each Hebrew construct chain: (1) identify the construct noun and its form, (2) identify the absolute noun, (3) determine whether the chain is definite or indefinite, (4) translate.

## Coverage

| Part | Description | Items |
|---|---|---|
| Part A | Simple 2-link chains — indefinite | 8 |
| Part B | Simple 2-link chains — definite (article or proper noun) | 8 |
| Part C | Extended 3-link chains | 5 |
| Part D | Chains with feminine construct nouns (ַת- ending) | 4 |
