# Ch8 Pronoun Identification Drill — Hebrew Pronouns

Identify, parse, and translate 25 Hebrew pronouns drawn from real and realistic biblical Hebrew phrases. Students classify each pronoun by type (Personal / Demonstrative / Relative / Interrogative), isolate the pronoun, parse it (PGN where applicable), and translate the full phrase.

## Coverage

| Part | Pronoun Type | Items | Description |
|---|---|---|---|
| Items 1–8 | Independent Personal | 8 | All major personal pronouns — 1cs (both אֲנִי and אָנֹכִי), 2ms, 3ms, 3fs, 1cp, 2mp, 3mp |
| Items 9–15 | Demonstrative | 7 | Near demonstratives (זֶה, זֹאת, אֵלֶּה) — attributive and predicate uses; far demonstratives (הַהוּא, הַהִיא) |
| Items 16–21 | Relative (אֲשֶׁר) | 6 | Subject relative, object relative, various noun antecedents |
| Items 22–25 | Interrogative | 4 | מִי (persons) and מַה (things) |
