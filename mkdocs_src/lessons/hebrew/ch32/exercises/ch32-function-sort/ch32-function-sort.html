BBH Chapter 32 · Pual Strong Verbs

The Pual is always the passive of the Piel. For each of the 20 Pual forms: (1) give the Piel active meaning; (2) translate the Pual passive form.

Pual diagnostic: Qibbuts (ֻ) or holem (ֹ) under R1 + Dagesh Forte in R2.
Note: When R2 is a guttural (ר, ח, ע, etc.), dagesh forte is rejected and the preceding vowel lengthens — qibbuts → holem (e.g., בֹּרַךְ not בֻּרַּךְ).

Pattern summary:
Perfect 3ms: qibbuts/holem under R1 + dagesh + patach in R2 (e.g., קֻטַּל, בֹּרַךְ)
Imperfect 3ms: יְ prefix + same pattern (e.g., יְקֻטַּל, יְבֹרַךְ)
Participle ms: מְ prefix + qibbuts/holem under R1 + dagesh + qamets in R2 (e.g., מְקֻטָּל, מְבֹרָךְ)
Perfect 3cp: qibbuts under R1 + shewa in R2 + וּ suffix (e.g., כֻּבְּדוּ, שֻׁלְּחוּ)
