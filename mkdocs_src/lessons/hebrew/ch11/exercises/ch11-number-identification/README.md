# Ch11 Number Identification Drill — Hebrew Numbers

For each of 20 Hebrew number-noun phrases: (1) identify the number word, (2) give its numeric value, (3) describe the gender polarity situation (if applicable), (4) translate into English.

## Coverage

| Part | Description | Items |
|---|---|---|
| Part A | Cardinal numbers 1–10 — mixed masculine and feminine nouns to test gender polarity; includes one ten (40) and a number 2 construct | 8 |
| Part B | Teens and additional cardinals — numbers 9, 12, 13, 15, 16, 30, 70; tests polarity in teen unit portions and invariable tens | 8 |
| Part C | Ordinal numbers 1st–10th — attributive adjective syntax; no gender polarity | 4 |
