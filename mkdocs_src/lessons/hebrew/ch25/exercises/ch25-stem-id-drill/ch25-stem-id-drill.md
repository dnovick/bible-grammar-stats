# Chapter 25 — Stem-ID Drill (Weak Roots)

*Identifying Qal and Niphal forms of III-ה, I-נ, and hollow roots*

---

## Instructions

For each Hebrew form below, identify:

1. **Stem** — Qal or Niphal
2. **Conjugation** — Perfect, Imperfect, Wayyiqtol, Imperative, Inf. Absolute, Participle
3. **PGN** — person/gender/number (write **—** for non-finite forms)
4. **Root** — the three-letter lexical root
