## Instructions

Write the correct Niphal form for each entry. Work from memory first; then check your answers below.

> **III-ה tip:** The final ה of the root behaves like a vowel letter throughout. Watch for the distinctive ֶה endings in the imperfect and participle.

> **I-נ tip:** The nun of the root assimilates into the following consonant via dagesh forte in the Niphal imperfect (יִפָּל, not *יִנְפָּל). In the perfect, the nun remains: נִפַּל.

## Answer Key

### Section A — גלה (III-ה) Niphal

| Form | Answer | Notes |
|---|---|---|
| Perfect 3ms | נִגְלָה | Final ָה = III-ה ending in perfect |
| Perfect 3fs | נִגְלְתָה | Feminine ָה suffix added to stem |
| Perfect 3cp | נִגְלוּ | 3cp ו suffix |
| Perfect 1cs | נִגְלֵיתִי | 1cs תִי suffix; ֵי appears before consonantal suffix |
| Imperfect 3ms | יִגָּלֶה | Dagesh forte in ג (assimilated נ) + ֶה ending |
| Imperfect 3fs | תִּגָּלֶה | Feminine prefix תּ |
| Imperfect 3mp | יִגָּלוּ | 3mp ו suffix drops the final ה |
| Imperfect 1cs | אֶגָּלֶה | 1cs prefix א |
| Imperative 2ms | הִגָּלֵה | הִ prefix + dagesh forte + ֵה ending |
| Inf. Construct | הִגָּלוֹת | Inf. Construct with ות suffix; הִ prefix |
| Inf. Absolute | הִגָּלֹה | הִ prefix + dagesh forte + ֹה ending |
| Participle ms | נִגְלֶה | נִ prefix + ֶה ending (III-ה participle) |
| Participle fs | נִגְלֵית | Feminine ית ending |

### Section B — נפל (I-נ) Niphal

| Form | Answer | Notes |
|---|---|---|
| Perfect 3ms | נִפַּל | נִ prefix; dagesh forte in פּ (the root nun does NOT assimilate in perfect) |
| Perfect 3fs | נִפְּלָה | 3fs ָה suffix; shewa under פ before suffix |
| Perfect 3cp | נִפְּלוּ | 3cp ו suffix |
| Imperfect 3ms | יִפָּל | Root nun assimilates: יִ + פָּ (dagesh forte) + ל |
| Imperfect 3fs | תִּפָּל | Feminine prefix תּ |
| Imperfect 3mp | יִפְּלוּ | 3mp ו suffix; shewa under פ |
| Imperative 2ms | הִפָּלֵל | הִ prefix + dagesh forte in פּ + ֵל ending |
| Participle ms | נֹפֵל | Unusual: I-נ Niphal participle takes נֹ (holem) under first root letter |

### Key Patterns Summary

| Feature | III-ה Niphal | I-נ Niphal |
|---|---|---|
| Perfect 3ms | נִגְלָה (ָה ending) | נִפַּל (dagesh in R2) |
| Imperfect 3ms | יִגָּלֶה (ֶה ending) | יִפָּל (root נ assimilates) |
| Participle ms | נִגְלֶה (ֶה ending) | נֹפֵל (holem under נ) |
| Imperative 2ms | הִגָּלֵה | הִפָּלֵל |
