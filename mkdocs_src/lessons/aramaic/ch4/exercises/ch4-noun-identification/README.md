# "Noun Identification Drill" — BBA Ch4 Absolute State

## Description
20-item identification drill covering absolute-state noun forms drawn from Chapter 4 vocabulary and closely related inflected forms. Students identify gender, number, state, and the root/lexical form (dictionary entry) for each given Aramaic noun.
