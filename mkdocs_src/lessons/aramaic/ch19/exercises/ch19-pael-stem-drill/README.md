# Exercise: Ch19 Pael Stem Drill

*Basics of Biblical Aramaic, Ch19 — The Pael Stem*

---

## Description

A 20-item parsing drill covering all conjugations of the **Pael stem** (D stem) using verbs from Daniel and Ezra. Students identify conjugation, root, PGN, and translation for each form.

---

## Conjugation Coverage

| Conjugation | Items | Verbs Used |
|---|---|---|
| Perfect | 1, 4, 5, 6, 9, 13, 16, 18 | שׁבח (3ms, 1cs, 3mp), ברך (3ms), חוה (3ms), שׁבח (1cp), קבל (3ms), שׁלח (3ms) |
| Imperfect | 2, 7, 10, 12, 15, 17, 20 | שׁבח (3ms, 3fs/2ms), ברך (3ms), חוה (3ms, 1cp), קבל (3mp), רבה (3ms) |
| Active Participle | 3, 8, 11 | שׁבח (ms), ברך (ms), חוה (ms) |
| Infinitive | 14 | שׁבח |
| Passive Participle | 19 | קטל (ms, model root) |

---
