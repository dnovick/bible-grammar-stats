# "Aramaic Vowel Identification" — Vowel Recognition Drill

## Description
20-item identification drill covering all major Aramaic vowel signs: long vowels, short vowels, compound shevas, and matres lectionis. Students identify the vowel name, class (long/short/reduced), transliteration, and whether a mater lectionis is present.
