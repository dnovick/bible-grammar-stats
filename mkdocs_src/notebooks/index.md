# Notebooks

Interactive analysis notebooks covering the full `bible_grammar` toolkit — Hebrew OT, Greek NT, Septuagint, Peshitta, and Targumim.

Each notebook below is rendered statically with its outputs. To run a notebook interactively, click the **Launch in Binder** badge on any notebook page, or launch the full environment:

[![Launch in Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/dnovick/berean-bible-bots/main?urlpath=lab/tree/notebooks/)

> **Note:** Binder sessions download the processed data files (~295 MB) at startup via `binder/postBuild`, so the first launch takes a few minutes.
