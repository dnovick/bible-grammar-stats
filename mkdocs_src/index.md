# Berean Bible Bots

Free exercises, flashcard decks, and reference materials for students of Biblical Hebrew, Aramaic, and Greek — grounded in real scripture data from the MACULA Hebrew WLC and Greek corpora.

> **⚠ AI-Generated / Work in Progress** — The code, lesson packages, exercises, Anki decks, and analysis outputs in this project were generated by [Claude](https://claude.ai) (Anthropic) under human direction. This is an experimental project under active development. You may encounter errors, incomplete content, or inaccurate linguistic analysis. Feedback and bug reports are welcome — please [open a GitHub issue](https://github.com/dnovick/berean-bible-bots/issues).

| [Biblical Aramaic](lessons/aramaic/index.md) | *Basics of Biblical Aramaic* | Ch1–22 |

## What Each Chapter Contains

- **Lesson** — full grammar reference with paradigms, examples, and frequency data
- **Exercises** — interactive HTML exercises (fillable fields, per-item answer reveal), static Markdown with answer key, and printable PDF
- **Flashcard decks** — Anki-compatible morphology and vocabulary decks
